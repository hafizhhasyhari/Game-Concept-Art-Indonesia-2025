## ğŸ¯ Tentang Repositori Ini

Selamat datang! Repositori ini adalah jurnal belajar pribadi saya untuk mata kuliah **Game Concept Art**. Tujuannya adalah untuk mendokumentasikan seluruh proses pembelajaran selama 16 minggu, dari ideasi 2D di Clip Studio Paint hingga pemodelan 3D yang kompleks di Maya dan Blender.

Repositori ini melacak semua 11 komponen pembelajaran: materi, praktik, latihan, pameran, proyek, penelitian, pengabdian, HAKI, catatan, nasihat, dan naskah.

### ğŸŒŸ Fokus Pembelajaran:
